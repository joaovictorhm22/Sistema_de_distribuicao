## Instalação :hammer_and_wrench:
Clone o repositorio do projeto 
```bash 
  git clone https://github.com/joaovictorhm22/Sistema_de_distribuicao.git
  cd Sistema_de_distribuicao
```
Compile o programa
```bash 
  g++ -o Main.cpp 
```
Execute o programa
```bash 
  ./Main.exe 
```
